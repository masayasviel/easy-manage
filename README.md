# easy-manage
簡単管理ツール

## phpMyAdmin

http://localhost:8080/

## Usage

```bash
cp example.env .env
```

## Docker

### Run

```bash
make up
```

### Down

```bash
make down
```

### Init DB

```bash
make init-db
```
